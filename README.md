# Core concept

Hệ thống đưa ra giải pháp tracking(theo dõi) và chăm sóc khách hàng từ bước tiếp cận khách hàng tới lúc chốt đơn, chế độ hậu mãi, chăm sóc sau bán cùng và upsale khách hàng
